# 🥗 NutriGo – Aplikasi Pemantau Gizi Harian

**NutriGo** merupakan aplikasi berbasis **desktop** yang dikembangkan menggunakan **bahasa pemrograman Java** dengan **Apache NetBeans IDE**. Aplikasi ini dirancang untuk membantu pengguna dalam memantau kondisi kesehatan serta mengelola kebutuhan gizi harian secara sederhana dan mudah digunakan.

Melalui aplikasi ini, pengguna dapat mengetahui kondisi berat badan, memperoleh rekomendasi menu makanan sehat, serta mendapatkan saran aktivitas olahraga yang dapat dilakukan untuk menjaga kebugaran tubuh. NutriGo diharapkan dapat menjadi media edukasi yang membantu meningkatkan kesadaran masyarakat akan pentingnya pola hidup sehat.

---

## 📌 Informasi Proyek

* **Nama Proyek** : NutriGo
* **Jenis Aplikasi** : Desktop Application
* **Bahasa Pemrograman** : Java
* **IDE** : Apache NetBeans
* **Framework GUI** : Java Swing

---

## 🎯 Tujuan Pengembangan

Pengembangan aplikasi **NutriGo** bertujuan untuk:

* Membantu pengguna memantau kondisi kesehatan secara sederhana.
* Memberikan informasi mengenai kebutuhan gizi harian.
* Memberikan referensi resep makanan sehat yang mudah dibuat.
* Mendorong pengguna untuk menerapkan gaya hidup sehat melalui olahraga dan pola makan seimbang.

---

## ⚙️ Fitur Utama Aplikasi

### 1️⃣ Perhitungan dan Pemantauan Berat Badan

Pengguna dapat mengetahui kondisi berat badan sebagai salah satu indikator kesehatan tubuh.

### 2️⃣ Informasi Gizi Harian

Aplikasi memberikan gambaran mengenai kebutuhan gizi yang diperlukan oleh tubuh untuk mendukung aktivitas sehari-hari.

### 3️⃣ Rekomendasi Resep Makanan Sehat

NutriGo menyediakan beberapa referensi resep makanan sehat yang dapat membantu pengguna dalam menjaga pola makan yang lebih baik.

### 4️⃣ Rekomendasi Aktivitas Olahraga

Aplikasi memberikan saran aktivitas olahraga sederhana yang dapat dilakukan untuk menjaga kebugaran dan kesehatan tubuh.
